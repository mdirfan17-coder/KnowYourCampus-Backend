# KnowYourCampus

KnowYourCampus is a Java backend application designed to provide students with a single platform to discover verified campus events, workshops, hackathons, placement drives, competitions, and other opportunities.

The project is being developed incrementally while learning Java backend concepts. Each new concept—such as OOP, Collections, File Handling, Exception Handling, and more—is integrated into the project using industry-inspired design practices.

## Vision

Instead of students relying on multiple WhatsApp groups, emails, and notice boards, KnowYourCampus aims to serve as a centralized platform where verified opportunities are easy to discover and manage.

- User Management
- Admin Management
- Event Management
